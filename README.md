# Greennes-HMI-Project
This repository contains the work of Axelle Briens-Solignac and Alice Lombard for the HMI course at INSA (2025-2026).

## Greennes: find closest bikes, parks and trashbin in Rennes being aware of the current weather conditions
Greennes is a web application that helps users find the nearest bikes, parks, and trash bins in Rennes while considering the current weather conditions. 

## For who?
This app is designed for residents and visitors of Rennes who want to navigate the city efficiently and eco-friendly while being mindful of the weather.

## Technologies
The app is built using VueJs and TypeScript.
